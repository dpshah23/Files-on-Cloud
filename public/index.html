📋 How to Use Files on Cloud

  • 🔐 Authentication: Create an account or login to upload files. Your files are securely associated with your account.
  • Upload Files: Click on the upload area to select your file (maximum 20MB). You can optionally provide a custom 5-digit code, set a password, and choose expiration time.
  • Password Protection: Optionally protect your files with a password. Recipients will need the password to download.
  • Expiration: Files automatically expire and are deleted after the chosen time period (1 hour to 7 days).
  • QR Codes: After upload, get a QR code that others can scan to access the download link instantly.
  • Share Code: After uploading, share the 5-digit code with others to allow them to download your file.
  • Download Files: Enter the 5-digit code you received and click "Download File" to get the file instantly.
  • Get File Info: Use "Get File Info" button to check file details like name, size, and download count before downloading.
  • Analytics: View download statistics for your uploaded files through your account.
  • Security: Files are temporarily stored and automatically deleted after expiration for your privacy and security.
  • File Types: All file types are supported - documents, images, videos, archives, and more!
